# Feature: Redesign SpecWeave Documentation Site

## Overview

The SpecWeave Docusaurus docs site has 186 files but only 100 are referenced in `sidebars.ts`. After excluding autogenerated sections (academy essentials, talks, glossary terms), **46 files are truly orphaned** -- invisible to users navigating the site. The dashboard has 16 pages in source code but only 4 are mentioned in a single doc page (`analytics-dashboard.md`) that is itself orphaned from the sidebar. The features page is a 605-line monolith. The Guides section is a catch-all bucket with 13 items in sidebar but 45+ orphaned guide files.

This increment restructures the sidebar navigation, trims the features monolith, creates dashboard capability documentation, and adds cross-linking -- all without moving or renaming any files (sidebar-only restructure for phases 1 and 4, new files only for phase 3).

**Target**: `repositories/anton-abyzov/specweave/docs-site/`
