# 0329: Homepage Video-First Redesign

**Type**: feature | **Priority**: P1 | **Status**: in-progress

## Overview
Redesign spec-weave.com homepage from 8 verbose sections to 5 sleek, world-class sections with `ship-while-you-sleep.mp4` video as the hero centerpiece. Inspired by Linear, Vercel, and Cursor landing pages.

## User Stories

### US-001: Video-First Hero
As a visitor, I want to see a cinematic video hero that immediately shows what SpecWeave does, so I understand the product in seconds.
- **AC-US1-01**: [x] ship-while-you-sleep.mp4 video is prominently displayed as the hero centerpiece
- **AC-US1-02**: [x] Video autoplays muted with loop
- **AC-US1-03**: [x] Clear headline "Ship Features While You Sleep" with gradient text
- **AC-US1-04**: [x] Two CTAs: "Get Started" and "Watch Demo"

### US-002: Consolidated Section Architecture
As a visitor, I want a streamlined page that communicates value quickly, not a wall of text.
- **AC-US2-01**: [x] Page has exactly 5 sections: VideoHero, Workflow, Capabilities, Trust, CTA
- **AC-US2-02**: [x] ProblemSection, VSkillsSection, WhatsNewSection, StatsSection removed
- **AC-US2-03**: [x] FeaturesSection + IntegrationsSection consolidated into Capabilities + Trust

### US-003: Key Differentiators Highlighted
As a visitor, I want to quickly understand what makes SpecWeave unique.
- **AC-US3-01**: [x] 6 capability cards: Multi-Agent Teams, Persistent Memory, Autonomous Execution, Quality Gates, Living Docs, Bidirectional Sync
- **AC-US3-02**: [x] 3-step workflow clearly shows the Define/Execute/Ship flow

### US-004: Brief Verified Skills Mention
As a visitor, I want to know about skill security without being overwhelmed.
- **AC-US4-01**: [x] Verified skills mentioned as compact callout (not full section)
- **AC-US4-02**: [x] Link to verifiedskill.com or verified skills docs

### US-005: World-Class Design
As a visitor, I want the page to feel premium and modern.
- **AC-US5-01**: [x] Dark cinematic hero section
- **AC-US5-02**: [x] Generous whitespace between sections
- **AC-US5-03**: [x] Responsive design (desktop, tablet, mobile)
- **AC-US5-04**: [x] Dark mode support via Docusaurus theme toggle
