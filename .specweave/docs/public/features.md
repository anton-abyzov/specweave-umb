|---------|---------|
| `/sw:increment "title"` | Create new increment |
| `/sw:do` | Execute implementation |

| Hook | Event | Default Actions |
|------|-------|-----------------|

## Configuration

### Config File (.specweave/config.json)

```json
{
  "sync": {
    "enabled": true,
    "activeProfile": "my-project",
    "settings": {
      "canUpsertInternalItems": true,
      "canUpdateExternalItems": true,
      "canUpdateStatus": true
    },
    "profiles": {
      "my-project": {
        "provider": "github",
        "config": { "owner": "org", "repo": "repo" }
      }
    }
  },
  "hooks": {
    "post_task_completion": {
      "sync_living_docs": true,
      "external_tracker_sync": true
    }
  },
  "limits": {
    "maxActiveIncrements": 1,
    "hardCap": 2
  }
}
```
