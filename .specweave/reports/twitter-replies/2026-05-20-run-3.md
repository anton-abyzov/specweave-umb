## The fix (still pending after run 1 and run 2)

Append to `~/.claude/settings.local.json` under `permissions.allow`:

```json
{
  "permissions": {
    "allow": [
      "mcp__Claude_in_Chrome__navigate",
      "mcp__Claude_in_Chrome__computer",
      "mcp__Claude_in_Chrome__find",
      "mcp__Claude_in_Chrome__form_input",
      "mcp__Claude_in_Chrome__browser_batch",
      "mcp__Claude_in_Chrome__tabs_create_mcp",
      "mcp__Claude_in_Chrome__tabs_context_mcp",
      "mcp__Claude_in_Chrome__javascript_tool",
      "mcp__Claude_in_Chrome__get_page_text",
      "mcp__Claude_in_Chrome__read_page"
    ]
  }
}
```

Restart Claude Code. The next scheduled run will actually post.

For a safer middle ground, scope the allow to x.com only:

```json
"mcp__Claude_in_Chrome__navigate(https://x.com/*)"
```

Same syntax for the other tools where applicable.
